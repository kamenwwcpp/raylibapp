# raylibapp

Простой проект на C++ с использованием [raylib](https://www.raylib.com/).

## Зависимости

- CMake 3.16+
- C++17 compiler (GCC, Clang, MSVC)
- Git

raylib загружается и собирается автоматически при конфигурации проекта.

## Сборка

### Linux

```sh
cmake -B build
cmake --build build
./build/raylibapp
```

### macOS

```sh
cmake -B build
cmake --build build
./build/raylibapp
```

### Windows

```sh
cmake -B build
cmake --build build
.\build\Debug\raylibapp.exe
```
